Napoleon Bonaparte

Website design for biogrphy of Napoleon Bonaparte

Feb 2017
Type: Web design
Platform: Website
Type of Work: Personal Project

This project is design a website for Napoleon Bonaparte biography and try to find represent typeface at that time. Biography website is always comlicated and hard to find information what we need. My design was focused on how users can see his personal life and his accomplishment at the same time. Ad using artworks about Napoleon Bonaparte, users can experience enjoyable finding informations.
